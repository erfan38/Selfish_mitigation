# 🛠️ **Selfish Mining Mitigation in PoW Blockchains**

## 📜 **Project Overview**

This project addresses **selfish mining attacks** in Proof-of-Work (PoW) blockchain networks. Selfish mining occurs when miners or groups of miners withhold newly mined blocks, maintaining a private chain to gain an unfair advantage. This behavior wastes honest miners' resources and degrades the overall efficiency of the network.

Our approach introduces a **game-theoretic solution** using a **war of attrition framework** to mitigate selfish mining. We design mechanisms that:

- Incentivize **honest mining** through **rewards**.
- Discourage selfish mining with **punishments**.
- Employ a **reputation function** to penalize miners who engage in selfish behavior.
- Use **evolutionary game theory** to analyze the stability of honest strategies.

Our findings indicate that honest mining remains stable under the proposed model, providing a practical solution to reduce selfish mining threats in blockchain networks.

## 🚀 **Getting Started**

### **Prerequisites**

- **Python 3.x** installed
- Required libraries (install via `pip`):

  ```bash
  pip install matplotlib numpy pandas
  ```

### **Run the Simulation**

1. Clone the repository:

   ```bash
   git clone https://github.com/erfan38/Selfish_mitigation.git
   cd Selfish_mitigation
   ```

2. Run the simulation script:

   ```bash
   python src/simulation.py
   ```

3. View the generated results in the `figures` folder.
